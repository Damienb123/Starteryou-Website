TextContent

Mongoose schema for the `TextContent` model. Represents the text content associated with a specific UI component.
Type:
  • Object
Properties:
Name Type Description
component string The name of the component this content belongs to. (Required)
content string The main content text for the component. Defaults to an empty string.
paragraphs Array.<string> An array of paragraphs for the component. Defaults to an empty array.
createdAt Date Timestamp indicating when the document was created. (Added by `timestamps`)
updatedAt Date Timestamp indicating when the document was last updated. (Added by `timestamps`)
