Module: MongoTester

MongoTester class to test MongoDB connection, database operations, and network connectivity.

This utility class helps in verifying MongoDB connection and performing basic operations like write, read, and cleanup. It also provides network connectivity tests to ensure that the MongoDB server is reachable from the application.
