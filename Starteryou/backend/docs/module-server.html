Module: server

Express server setup for handling file management and system verification routes.

This script initializes an Express server, connects to MongoDB, sets up routes for file management (upload, download, etc.), and system verification (file integrity checks). It also includes error handling, connection retries, logging, and graceful shutdown functionality.
