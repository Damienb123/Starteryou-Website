(static, constant) API_CONFIG :Object

Central configuration object for API interactions. Includes base URL and endpoint definitions for all file operations.

Usage examples:

// Fetch file by title
fetch(`${API_CONFIG.baseURL}${API_CONFIG.endpoints.fileByTitle('example')}`)

// Upload new file
fetch(`${API_CONFIG.baseURL}${API_CONFIG.endpoints.fileUpload}`, {
  method: 'POST',
  body: formData
})

// Update existing file
fetch(`${API_CONFIG.baseURL}${API_CONFIG.endpoints.fileUpdate('example')}`, {
  method: 'PUT',
  body: formData
})
